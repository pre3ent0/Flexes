flex-flow: row nowrap;

Назначение: лень хуярить по 2 элемента? используй один, вуаля

justify-content: flex-start;

Назначение: расположите элементы по основной оси

Возможные значение: flex-start / flex-end / center / space-between / space-around

align-items: flex-start;

Назначение: расположите элементы по вертикали

Возможные значение: flex-start / flex-end / center / baseline / stretch

align-content: stretch;

Назначение: выравнивает строки элементов по вертикали

Возможные значение: flex-start / flex-end / center / space-around / stretch / space-between
